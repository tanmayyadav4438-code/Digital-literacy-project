Task 3: Coding & Collaboration Platforms

This folder contains my practical work for Task 3, which focuses on exploring essential online platforms for coding and team collaboration.

For the coding practice component, I set up an account on HackerRank. Building a strong foundation in logic and algorithms is critical for developing efficient AI and machine learning models later on, so I started by completing the introductory Python challenges. The platform's integrated environment helped me practice writing and testing code directly in the browser. You will find the screenshot of my completed challenge and badge saved in this folder.

For the collaboration component, I used Google Forms to build a 5-question "Digital Literacy Awareness Quiz." I mixed multiple-choice and short-answer formats to test basic internet safety knowledge. I also linked this form to a Google Sheet to automatically collect and organize the responses. Screenshots of both the form and the spreadsheet are included here.

These platforms will be incredibly useful throughout my degree. HackerRank gives me a structured way to continuously sharpen my coding skills for future technical interviews. Meanwhile, Google Forms and Sheets will make it much easier to collaborate with my batchmates, collect survey data for research projects, and organize group work smoothly.
