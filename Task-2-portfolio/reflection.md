Task 2: Building a Student Digital Portfolio

This reflection note documents my progress for Task 2, where I built my foundational digital portfolio. A professional digital presence is essential today, so I created and updated accounts on three key platforms: GitHub, LinkedIn, and Kaggle.

I chose GitHub because it is the primary platform for developers to host code. I set up my profile README with my name, my branch in Computer Science and Engineering with a specialization in AI and ML, my current year, and my learning goals. I selected LinkedIn to build my professional network, ensuring my Education section reflects my current degree and expected graduation year. Lastly, I chose Kaggle because it is the perfect environment for practicing data science and training machine learning models.

Over the next four years, I plan to use GitHub actively to store the code for all my Python and AI/ML hackathon projects. I will use LinkedIn to connect with industry professionals, showcase my deployed websites, and eventually look for internships. Kaggle will serve as my main practice area for improving my predictive modeling skills. By keeping these profiles professional and minimal at this stage, I am building a strong digital footprint for my future career.
